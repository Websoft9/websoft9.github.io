# 关于

本文档由 Websoft9 公司提供，用于指导用户学习 Scratch3.0 的安装部署与基本运维技术，包括：Scratch部署、初始化安装、环境配置、HTTPS、SMTP、备份升级和连接云服务器操作等基本操作。

Scratch 是一款由麻省理工学院(MIT) 设计开发的一款面向少年的简易编程工具，scratch已经是少儿编程行业的基础软件。使用 Scratch，你可以编写属于你的互动媒体，像是故事、游戏、动画，然后你可以将你的创意分享给全世界。Scratch3.0是一个部署在服务器上的Web版本，有浏览器就可以使用Scratch。
