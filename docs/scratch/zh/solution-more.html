# 域名绑定

绑定域名的前置条件是：Scratch已经可以通过解析后的域名访问。

虽然如此，从服务器安全和后续维护考量，域名绑定步骤不可省却

Scratch 域名绑定操作步骤：

  1. 登录云服务器
  2. 修改 Nginx虚拟机主机配置文件，将其中的server_name的值 _ 修改为你的域名
     server {
 listen 80;
 server_name _;   # _改为自定义域名
 root /data/wwwroot/scratch-gui/build;
 }
  3. 保存配置文件，重启Nginx服务
