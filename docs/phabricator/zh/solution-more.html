# 域名绑定

绑定域名的前置条件是：Phabricator已经可以通过解析后的域名访问。

虽然如此，从服务器安全和后续维护考量，域名绑定步骤不可省却

Phabricator 域名绑定操作步骤：

  1. 登录云服务器
  2. 修改 Apache虚拟机主机配置文件，将其中的 ServerName 项的值 www.example.com 修改为你的域名
     <VirtualHost *:80>
 ServerName www.example.com
 ...
 </VirtualHost>
  3. 保存配置文件，重启Apache服务
