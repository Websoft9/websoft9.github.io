# 安装网站示例

安装完整的经典步骤包括：①上传网站代码->②修改文件系统用户权限->③配置域名（非必要）->④增加网站对应的数据库（非必要）->⑤完成安装向导

在BT面板中，基本遵循以上原则（步骤先后顺序略有差异）

# 示例：安装WordPress

具体操作如下：

  1. 打开宝塔Web界面-->网站-->添加站点，完成必要的配置，并牢记相关账号和密码

    • 域名必须填写
    • 建议创建FTP（如不创建，通过使用宝塔的在线文件管理）
    • 创建数据库
    • 选择程序和版本

  2. 上传文件到网站目录或上传压缩包到目录后解压；

  3. 到云控制台中完成域名解析

  4. 打开网址 http://您的域名 ,即可开始进行wordpress配置。

# FAQ

# 添加站点，提示PHP版本不能为空？

问题描述：当只安装一个PHP版本的时候，增加网站会提示“PHP版本不能为空”，这是系统的一个Bug 解决方案：再安装一个PHP版本即可
