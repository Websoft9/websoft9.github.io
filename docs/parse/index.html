# Overview

This documentation is provided by Websoft9, it can help users to acquire skills of Parse Server Deployment, Installation, Configuration, HTTPS, SMTP, Backup, Upgrade, Connect Instance and more.

Parse Server is an open source backend or an API server module that can be deployed to any infrastructure that can run Node.js.
