# MySQL

LAMP deployment package includes MySQL and Web-GUI tool phpMyAdmin for MySQL management, refer to these steps to use it:

  1. Using the Chrome or FireFox to visit URL http://Internet IP/phpmyadmin
  2. Enter username and password of MySQL(Don't known password?)
  3. Start to manage MySQL now

Websoft9 provide MySQL guide for more useful skills of MySQL, includes: modify password, create user, import/export data, enable or disable remote visit, log configuation and so on.
