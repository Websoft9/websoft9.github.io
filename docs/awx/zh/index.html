# 关于

本文档由 Websoft9 公司提供，用于指导用户学习 AWX 的安装部署与基本运维技术，包括：AWX部署、初始化安装、环境配置、HTTPS、SMTP、备份升级和连接云服务器操作等基本操作。

AWX 是Ansible Tower的开源版，Ansible Tower是一个可视化界面的服务器自动部署和运维管理平台。AWX提供基于Web的用户界面，REST API和构建在Ansible之上的任务引擎。
