# 关于

本文档由 Websoft9 公司提供，用于指导用户学习 RabbitMQ 的安装部署与基本运维技术，包括：RabbitMQ部署、初始化安装、环境配置、HTTPS、SMTP、备份升级和连接云服务器操作等基本操作。

RabbitMQ 是Apache出品，流行的、能力强劲的开源消息总线，完全支持JMS1.1和J2EE 1.4规范（持久化，XA消息，事务)，尽管JMS规范出台已经是很久的事情了，但是JMS在当今的J2EE应用中间仍然扮演着特殊的地位。RabbitMQ支持多种语言和协议编写客户端。
