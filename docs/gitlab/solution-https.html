# SSL/HTTPS

HTTPS can be set up after the completed the domain name binding and can be accessed via HTTP.

The GitLab Omnibus package is a highly integrated, configurable environment, and the HTTPS settings are not the same as the usual Nginx installation. Refer to the official documentation for specific reference: Enabling HTTPS
