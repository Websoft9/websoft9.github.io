# Overview

This documentation is provided by Websoft9, it can help users to acquire skills of Nextcloud Deployment, Installation, Configuration, HTTPS, SMTP, Backup, Upgrade, Connect Instance and more.

Nextcloud is the next generation open source Enterprise File Sync and Share was started by ownCloud inventor Frank Karlitschek and a dozen experienced open source entrepreneurs and engineers to empower users to take back control over their data and communication.
