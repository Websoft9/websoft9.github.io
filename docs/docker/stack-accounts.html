# Username and Password

These accounts are required for Docker image installation and configuration

# Portainer

The administrator username and password is created by yourself when load Portainer at first time

# Linux

  • Host Name: Internet IP or Public IP of your Instance
  • Connect by: Online SSH on Cloud Console or SFTP/SSH tools on your local computer
  • Password: It was set by yourself when created instance
  • Username: Different Cloud Platform has differences
    Cloud Platform Administrator Username Other
    Azure It was set by yourself when created instance How to enable root access?
    AWS CentOS centos How to enable root access?
    AWS Ubuntu ubuntu How to enable root access?
    Alibaba Cloud, HUAWEI CLOUD, Tencent Cloud root

If don't remember the password of Linux, you should reset password on Cloud Console
