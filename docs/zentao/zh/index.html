# 关于

本文档由 Websoft9 公司提供，用于指导用户学习 ZenTao（禅道） 的安装部署与基本运维技术，包括：ZenTao 部署、初始化安装、环境配置、HTTPS、SMTP、备份升级和连接云服务器操作等基本操作。

ZenTao（禅道） 由青岛易软天创网络科技有限公司（用心做开源的杰出公司）开发，是一款优秀的开源项目管理软件。它集产品管理、项目管理、质量管理、文档管理、组织管理和事务管理于一体，是一款专业的研发项目管理软件，完整覆盖了研发项目管理的核心流程。禅道管理思想注重实效，功能完备丰富，操作简洁高效，界面美观大方，搜索功能强大，统计报表丰富多样，软件架构合理，扩展灵活，有完善的API可以调用。
