# 关于

本文档由 Websoft9 公司提供，用于指导用户学习 Seafile 的安装部署与基本运维技术，包括：Seafile部署、初始化安装、环境配置、HTTPS、SMTP、备份升级和连接云服务器操作等基本操作。

Seafile 是一款开源的企业云盘，注重可靠性和性能。支持 Windows, Mac, Linux, iOS, Android 平台。支持文件同步或者直接挂载到本地访问。私有云盘产品 Seafile 起源于创始人清华实验室时期，历经6年的打磨，已发展成为一个国际化的开源项目，在 GitHub 上的项目有超过4500人关注，在国内开源社区开源中国上面也赢得了很多赞誉。
