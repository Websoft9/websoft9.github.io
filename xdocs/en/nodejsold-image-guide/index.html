How to change Node.js version

1. Use NVM to list available Node.js versions:

nvm ls-remote

2. Enter nvm ls to display the locally installed version of the Node.js, as well as the version currently in use.

3. If you need to install Node.js v7.7.3， enter nvm install v7.7.3 to install it locally.

4. When you install Node.js, you can enter node --version to confirm the current version.

5. You can use nvm use 7.7.3  to specify the version you want to use, or use nvm alias default 7.7.3 to specify a permanent version. Then you can enter node --version to confirm the current version.
