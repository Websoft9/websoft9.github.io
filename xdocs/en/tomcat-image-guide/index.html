Environment description

After loading the image, To access, Using following URL: http://< Public net IP >

Software version: Tomcat8, JAVA1.8.0_121
Tomcat install directory: /usr/local/tomcat
Tomcat commands: /usr/local/tomcat/bin
Tomcat Configuration File: /usr/local/tomcat/conf
Web Context document root for Tomcat: /data/wwwroot/default
To access the deault url for tomcat: http://< Public net IP >:8080

How to manage service

Tomcat: service tomcat {start|stop|status|restart|reload}  

Nginx: service nginx {start|stop|status|restart|reload|configtest}
MySQL: service mysqld {start|stop|restart|reload|status}
PHP: service php-fpm {start|stop|restart|reload|status}
Pure-Ftpd: service pureftpd {start|stop|restart|status}
Redis: service redis-server {start|stop|status|restart|reload}
Memcached: service memcached {start|stop|status|restart|reload}
